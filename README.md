# 🐍 Python Task Manager Project

A Python-based web application with multiple authentication methods, task management features, and contacts CRUD operations.

---

## ✨ Features

### 🔐 Authentication
- **Google Login** (OAuth2)
- **Social Media Login** (Facebook, GitHub, etc.)
- **Custom Login** with Email & Password

### ✅ Todo Task Management
- Create new tasks
- Update existing tasks
- Mark tasks as complete
- Close/delete tasks

### 📇 Contacts Management
- Create new contacts
- Read/view contacts list
- Update contact details
- Delete contacts
